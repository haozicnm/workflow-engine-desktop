# NNNN-决策标题

> 状态：[proposed | accepted | deprecated | superseded]
> 日期：YYYY-MM-DD
> 决策者：[谁做的决策]

## Context

描述决策的背景。是什么问题或机会促使了这个决策？

## Decision

描述做出的决策。用清晰的语言说明选择了什么方案。

## Consequences

描述决策的后果。包括：
- 正面影响
- 负面影响
- 需要的后续工作

## Alternatives Considered

列出考虑过的其他方案，以及为什么没有选择它们。

- 相关文档链接
- 相关 Issue 或 PR
