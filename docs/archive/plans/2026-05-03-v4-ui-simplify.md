# v4.0 UI 精简计划

## 删
- ❌ 顶部标签栏（WorkflowTabs）
- ❌ 左侧菜单栏（SideToolbar）
- ❌ 底部边栏
- ❌ 小地图（MiniMap）
- ❌ 属性面板（PropertyPanel）
- ❌ 顶部元素拾取按钮
- ❌ 录制功能（前端 + 后端 recording.rs）

## 留
- ✅ 顶部：工作流名称（点击可重命名）+ 运行 + 保存至工作流
- ✅ 控制台（通过右键菜单打开）
- ✅ 预览窗口（选中容器节点时右侧浮出）

## 缩
- 容器 action 参数 → 点击展开内联编辑（折叠面板）
- 节点加入 → 双击画布 / 右键菜单
- 定时任务 / 历史记录 → Dashboard

## 加
- Browser 容器新增 `pick`（拾取元素）action
- 右键菜单新增"控制台"入口
